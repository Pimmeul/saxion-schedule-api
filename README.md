## Endpoints
- [ ] **Auth**
  - [ ] /api/v1/auth/login [POST]
  - [ ] /api/v1/auth/register [POST]
  - [ ] /api/v1/auth/logout [POST]
- [ ] **Class**
  - [ ] /api/v1/class/?={search} [GET]
- [ ] **Lesson**
  - [ ] /api/v1/lesson/{classId}/{year}/{week} [GET]
- [ ] **User**
  - [ ] /api/v1/user/calendar [POST, GET, UPDATE, DELETE]

More endpoints to come.
